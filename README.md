## Install nvim

Ubuntu

    sudo add-apt-repository ppa:neovim-ppa/unstable
    sudo apt-get update
    sudo apt-get install neovim

macOS

    Download nvim-macos.tar.gz
    Extract: tar xzvf nvim-macos.tar.gz
    Run ./nvim-osx64/bin/nvim

Windows

    Download nvim-win64.zip
    Extract the zip.
    Run nvim-qt.exe
